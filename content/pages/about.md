## About This Site

This is a simple, minimalist blog built with modern tools:

- **11ty** - A simpler static site generator
- **ungrid.css** - A super lightweight responsive grid system
- **Markdown** - Simple content format

### What is ungrid?

[ungrid](https://chrisnager.com/ungrid/) is a minimal responsive grid system that weighs just 54 bytes when minified. It uses CSS display properties to create a flexible grid layout that works across all devices.

### Why minimalism?

A minimalist approach to web design means:
- Faster page loads
- Better accessibility
- Focus on content
- Easier to maintain
- Works everywhere

This site embraces these principles to provide a clean reading experience without unnecessary bloat.
